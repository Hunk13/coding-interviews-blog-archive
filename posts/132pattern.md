# 132 Pattern

## –ó–∞–¥–∞—á–∞

–î–∞–Ω –º–∞—Å—Å–∏–≤ –∏–∑ n —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª nums. –ü–∞—Ç—Ç–µ—Ä–Ω 132 ‚Äî —ç—Ç–æ –ø–æ–¥–ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ç—Ä–µ—Ö —á–∏—Å–µ–ª `nums[i]` , `nums[j]` –∏ `nums[k]` , —Ç–∞–∫–∏—Ö —á—Ç–æ `i < j < k` –∏ `nums[i] < nums[k] < nums[j]` .

–í–µ—Ä–Ω—É—Ç—å true, –µ—Å–ª–∏ –≤ nums –µ—Å—Ç—å –ø–∞—Ç—Ç–µ—Ä–Ω 132, –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –≤–µ—Ä–Ω—É—Ç—å false.

### –í–æ–ø—Ä–æ—Å—ã

1. **–£—Ç–æ—á–Ω–µ–Ω–∏–µ –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö:**
   - > –ú–æ–∂–µ—Ç–µ –ª–∏ –≤—ã —É—Ç–æ—á–Ω–∏—Ç—å —Ñ–æ—Ä–º–∞—Ç –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö? –≠—Ç–æ —Å–ø–∏—Å–æ–∫, –º–∞—Å—Å–∏–≤, —Å—Ç—Ä–æ–∫–∞ –∏–ª–∏ –¥—Ä—É–≥–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö?
   

     –ú–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±–∞—è –¥—Ä—É–≥–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä–∞—è —Å–º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å n —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª ‚Äî –Ω–µ –≤–∞–∂–Ω–æ, –≥–ª–∞–≤–Ω–æ–µ —á—Ç–æ–±—ã —Ñ—É–∫–Ω—Ü–∏—è —Ä–∞–±–æ—Ç–∞–ª–∞ –ø—Ä–∞–≤–∏–ª—å–Ω–æ. –°–∫–∞–∂–µ–º, –µ—Å–ª–∏ –ø–∏—à–µ–º –Ω–∞ –ø–ª—é—Å–∞—Ö, –ø—É—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏—è –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –≤–µ–∫—Ç–æ—Ä –ø–æ —Å—Å—ã–ª–∫–µ, –∞ –Ω–µ —É–∫–∞–∑–∞—Ç–µ–ª—å –Ω–∞ –Ω–∞—á–∞–ª–æ –º–∞—Å—Å–∏–≤–∞, –¥–ª—è —É–¥–æ–±—Å—Ç–≤–∞.

   - > –í—Å–µ–≥–¥–∞ –ª–∏ —ç–ª–µ–º–µ–Ω—Ç—ã –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö —è–≤–ª—è—é—Ç—Å—è —Ü–µ–ª—ã–º–∏ —á–∏—Å–ª–∞–º–∏? –ú–æ–≥—É—Ç –ª–∏ –æ–Ω–∏ –±—ã—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º–∏ –∏–ª–∏ –Ω—É–ª–µ–≤—ã–º–∏?

    
     –¶–µ–ª—ã–µ —á–∏—Å–ª–∞ `-10^9 <= nums[i] <= 10^9`

2. **–û–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è –∑–∞–¥–∞—á–∏:**
   - > –ö–∞–∫–æ–≤ –¥–∏–∞–ø–∞–∑–æ–Ω –∑–Ω–∞—á–µ–Ω–∏–π –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö? –ù—É–∂–Ω–æ –ª–∏ —É—á–∏—Ç—ã–≤–∞—Ç—å –∫—Ä–∞–π–Ω–∏–µ —Å–ª—É—á–∞–∏ —Å —á—Ä–µ–∑–º–µ—Ä–Ω–æ –±–æ–ª—å—à–∏–º–∏ –∏–ª–∏ –º–∞–ª–µ–Ω—å–∫–∏–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏?

     –ö–æ–ª–∏—á–µ—Å—Ç–≤–æ —á–∏—Å–µ–ª `1 <= n <= 2 * 10^5`

   - > –ï—Å—Ç—å –ª–∏ –∫–∞–∫–∏–µ-–ª–∏–±–æ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è –ø–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—é –ø–∞–º—è—Ç–∏ –∏–ª–∏ –≤—Ä–µ–º–µ–Ω–∏?

     –ö–∞–∫ –º–∏–Ω–∏–º—É–º, –¥–æ–ª–∂–Ω–æ —Ä–∞–±–æ—Ç–∞—Ç—å –∑–∞ –∫–≤–∞–¥—Ä–∞—Ç, –∞ –≤ –∏–¥–µ–∞–ª–µ ‚Äî –∑–∞ –ª–∏–Ω–∏—é. –ü–æ –ø–∞–º—è—Ç–∏ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–π –Ω–µ—Ç (–≤ –ø—Ä–µ–¥–µ–ª–∞—Ö —Ä–∞–∑—É–º–Ω–æ–≥–æ üòÖ)

3. **–û–∂–∏–¥–∞–µ–º—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç:**
   - > –ß—Ç–æ –¥–æ–ª–∂–Ω–∞ –≤–µ—Ä–Ω—É—Ç—å —Ñ—É–Ω–∫—Ü–∏—è, –µ—Å–ª–∏ –≤—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –ø—É—Å—Ç—ã –∏–ª–∏ —Ä–∞–≤–Ω—ã null?

     
     –°—á–∏—Ç–∞–µ–º, —á—Ç–æ —Ñ—É–Ω–∫—Ü–∏—è –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –≥–∞—Ä–∞–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ –Ω–µ–ø—É—Å—Ç–æ–π —Å–ø–∏—Å–æ–∫ —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª. –ü—Ä–æ –≤–∞–ª–∏–¥–∞—Ü–∏—é –Ω–µ –¥—É–º–∞–µ–º.

4. **–ê–ª–≥–æ—Ä–∏—Ç–º–∏—á–µ—Å–∫–∏–π –ø–æ–¥—Ö–æ–¥:**
   - > –ï—Å–ª–∏ –ø—Ä–∏–¥–µ—Ç—Å—è –≤—ã–±–∏—Ä–∞—Ç—å, —á—Ç–æ –≤–∞–∂–Ω–µ–µ –≤ –¥–∞–Ω–Ω–æ–π –∑–∞–¥–∞—á–µ ‚Äî –≤—Ä–µ–º—è –∏–ª–∏ –ø–∞–º—è—Ç—å?

     –í—Ä–µ–º—è –≤–∞–∂–Ω–µ–µ, –ø–∞–º—è—Ç–∏ –∑–∞–≤–∞–ª–∏—Å—å.

–ó–∞–¥–∞–≤–∞—è –ø—Ä–æ–¥—É–º–∞–Ω–Ω—ã–µ –≤–æ–ø—Ä–æ—Å—ã, –∫–∞–Ω–¥–∏–¥–∞—Ç –Ω–µ —Ç–æ–ª—å–∫–æ —Å–∞–º –ø–æ–Ω–∏–º–∞–µ—Ç –∑–∞–¥–∞—á—É, –Ω–æ –∏ –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç –º–µ—Ç–æ–¥ —Ä–µ—à–µ–Ω–∏—è –ø—Ä–æ–±–ª–µ–º –∏–Ω—Ç–µ—Ä–≤—å–µ—Ä—É.

### –ü—Ä–∏–º–µ—Ä—ã

–¢—Ä–∏ –ø—Ä–∏–º–µ—Ä–∞ –æ—Ç –∏–Ω—Ç–µ—Ä–≤—å—é—Ä–∞ —Å –æ–±—ä—è—Å–Ω–µ–Ω–∏—è–º–∏. –ù–∞ —ç—Ç–æ–º –º–µ—Å—Ç–µ –∏–Ω—Ç–µ—Ä–≤—å—é–µ—Ä —É–∂–µ –Ω–∞—á–∏–Ω–∞–µ—Ç –ø—Ä–æ—Å—ã–ø–∞—Ç—å—Å—è –∏ –ø–æ–Ω–∏–º–∞—Ç—å, —á—Ç–æ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –∏–Ω—Ç–µ—Ä–µ—Å–Ω–æ.

* –í –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –Ω–µ—Ç –ø–∞—Ç—Ç–µ—Ä–Ω–∞.

3. **–ü—Ä–∏–º–µ—Ä 4:**

* find132pattern([3, 5, 0, 3, 4]) -> true
* –ü–æ—è—Å–Ω–µ–Ω–∏–µ: –í –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –µ—Å—Ç—å –ø–∞—Ç—Ç–µ—Ä–Ω: [3, 5, 4]

–í–æ—Ç –∏ –æ—Ç–≤–µ—Ç ‚Äî —Ä–µ—á—å –ø—Ä–æ –ª—é–±—ã–µ —Ç—Ä–∏ —á–∏—Å–ª–∞ –≤ –º–∞—Å—Å–∏–≤–µ (–ø–æ–¥–ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å), –∞ –Ω–µ —Ç—Ä–∏ _–∏–¥—É—â–∏—Ö –ø–æ–¥—Ä—è–¥_ —á–∏—Å–ª–∞ (–ø–æ–¥–º–∞—Å—Å–∏–≤). –≠—Ç–æ –¥–µ–ª–∞–µ—Ç –∑–∞–¥–∞—á—É –Ω–µ—Ç—Ä–∏–≤–∏–∞–ª—å–Ω–æ–π.

* –ò—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç—Ä–∏ –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö —Ü–∏–∫–ª–∞ –¥–ª—è –≤—Å–µ—Ö –≤–æ–∑–º–æ–∂–Ω—ã—Ö –∫–æ–º–±–∏–Ω–∞—Ü–∏–π —Ç—Ä–æ–µ–∫ —á–∏—Å–µ–ª –≤ –º–∞—Å—Å–∏–≤–µ

  + –ï—Å–ª–∏ —ç—Ç–æ —É—Å–ª–æ–≤–∏–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –¥–ª—è —Ö–æ—Ç—è –±—ã –æ–¥–Ω–æ–π –∫–æ–º–±–∏–Ω–∞—Ü–∏–∏, –≤–µ—Ä–Ω—É—Ç—å `true`
